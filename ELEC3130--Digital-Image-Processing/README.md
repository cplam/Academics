# ELEC3130--Digital-Image-Processing
This is for the digital image processing course in HKUST.

## Course description:
This course introduces methods to process images on a computer. Topics include the formation and quantification of digital images, morphological image processing, image enhancement in the spatial and frequency domain, image restoration, color image processing, image compression, image segmentation, object recognition and face detection.
